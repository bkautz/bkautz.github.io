1. What is Git?

It is a free and open source version control software which very readily handles the management of changes and updates to small or large projects, documents, and computer programs quickly and efficiently.

2. Who started Git?

Git was started in 2005 by a man named Linus Torvalds who was a Finnish-American software engineer and hacker known for creating the Linus Kernel which became the foundation for Android and Chrome OS operating systems.

3. What is Hub?

In this case, it is a networking tool that allows multiple users or teams to collaborate, edit, store, and update changes to a program.

4. Describe Repository and it's role?

It is a storage area for projects and programs that gives access to other programmers by sending your GitHub link to them and they can view it right from the web browser.

5. My thoughts about GitHub.

I like that it's centralized storage place for my code or programs and that I can share it by sending my friends or fellow colleagues a link from GitHub. I may use it more.
